# Thought Topology Interface

## What is this project about?
This project lets you turn your thoughts into interactive graphs using only your hand gestures and a webcam. By tracking your hand movements, you can create, move, and connect nodes on a digital canvas—making it easy to visually organize and explore your ideas in real time.
